# Surveys and Benchmarks

## Memory in the Age of AI Agents (Survey)
- **Paper**: arXiv 2512.13564, December 2025 (107 pages)
- **Taxonomy**: Forms (token-level, parametric, latent) x Functions (factual, experiential, working) x Dynamics (formation, evolution, retrieval)
- Evolution = consolidation + updating + forgetting

## MemoryAgentBench (ICLR 2026)
- **Paper**: arXiv 2507.05257
- **Four competencies**: Accurate retrieval, test-time learning, long-range understanding, selective forgetting
- **Key finding**: No current system masters all four. Most fail on selective forgetting.

## Other Notable Papers
- **FadeMem** (arXiv 2601.18642) — Biologically-inspired dual-layer forgetting with Ebbinghaus curves
- **AgeMem** (arXiv 2601.01885) — RL-trained memory operations as callable tools
- **SymRAG** (arXiv 2506.12981) — Neuro-symbolic adaptive query routing
- **SimpleMem** (arXiv 2601.02553) — Efficient lifelong memory
